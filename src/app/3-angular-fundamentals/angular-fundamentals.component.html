Binding & template syntax

A quick playground for interpolation, pipes, bindings, and events.

Interpolation & expressions

Render values, expressions, and method output.

Title: {{ title }}

Uppercase: {{ title | uppercase }}

Greeting: {{ getGreeting() }}

Total: {{ price * quantity }}

User: {{ firstName + " " + lastName }}

Enabled: {{ isActive ? "yes" : "no" }}

{{ title }}
{{ title | uppercase }}
{{ getGreeting() }}
{{ price * quantity }}
{{ firstName + " " + lastName }}
{{ isActive ? "yes" : "no" }}

Property & attribute binding

Bind DOM properties and attributes.

Attribute colspan: {{ colSpan }} 
<h4 [textContent]="title"></h4>
<img [src]="imgUrl2" />
<td [attr.colspan]="colSpan">...</td>

Class & style binding

Toggle classes and inline styles.

<button [class.active]="isActive">...</button>
<button [style.backgroundColor]="isActive ? '#0f4c5c' : '#ef4444'">...</button>

Event binding

Handle click events and bubbling.

Click link 
<button (click)="sayHello($event)">...</button>
<div (click)="sayHello2($event)">...</div>
<a (click)="anchorClicked($event)">...</a>

Keyboard & template refs

Filter events and read input values.

<input (keyup)="keyUpHandler($event)" />
<input (keyup.enter)="enterHandler($event)" />
<input #inputValue (keyup.enter)="enterHandler2(inputValue.value)" />

Pipes

Format values for display.

Json: {{ person | json }}

Upper: {{ name | uppercase }}

Date: {{ createdAt | date: "yyyy-MM-dd" }}

Price: {{ amount | currency: "USD" }}

Percent: {{ ratio | percent: "1.0-2" }}

Short time: {{ dateObj | date: "shortTime" }}

Number: {{ numberValue | number: "3.1-5" }}

Persian: {{ dateObj | persianDate: "fa-IR" }}

{{ name | uppercase }}
